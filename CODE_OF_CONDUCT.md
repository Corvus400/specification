# Code of Conduct

このリポジトリは個人のポートフォリオ用仕様書リポジトリです。
公開された Issue、Pull Request、コメント、その他のやり取りでは、
相手を尊重し、職業的な場にふさわしいふるまいをしてください。

## Standards

許容される行動:

- 事実に基づいて、簡潔で再現可能な情報を共有する。
- 公開してよい情報だけを使って、リポジトリ運用上の問題を報告する。
- セキュリティやプライバシーに関わる内容を public Issue に投稿しない。

許容されない行動:

- ハラスメント、侮辱、差別的な発言、個人攻撃。
- 他者または自分の秘密情報、認証情報、アクセストークン、private URL、
  ローカルマシンの絶対パス、個人メールアドレス、非公開の個人情報の公開。
- 脆弱性の詳細、攻撃手順、再現用 exploit を public Issue、Pull Request、
  コメント、ログ、スクリーンショットに投稿すること。
- サポート窓口として Issue を使うこと。

## Enforcement

リポジトリ owner は、この Code of Conduct に反する Issue、Pull Request、
コメント、その他の投稿を close、hide、delete、または report することがあります。

脆弱性の疑いがある内容は GitHub Security Advisories から非公開で報告してください。
public Issue には投稿しないでください。
